# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 요구 사항
- 배트
    - [ ]  서로 다른 숫자 3개를 입력한다.

- 볼
    - [ ]  숫자를 랜덤하게 중복없이 3개 추출한다.
    
- 맞침
    - [ ] 같은 수가 같은 자리에 있으면 스트라이크를 출력한다.
    - [ ] 같은 수가 다른 자리에 있으면 볼을 출력한다.
    - [ ] 하나도 맞지않으면 낫싱을 출력한다.
    - [ ] 숫자가 모두 맞으면 종료된다.
